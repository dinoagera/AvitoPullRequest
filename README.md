Тестовое задание для Avito

Сервис автоматически назначает до 2 ревьюверов из команды автора при создании Pull Request. 
Поддерживает:
Управление командами и участниками
Создание, мерж и переназначение PR
Получение списка PR, где пользователь назначен ревьювером
Полная изоляция окружения через Docker


Перед этим нужно переимновать env.example -> .env
Запуск происходит через команду:
make up
make up
Остановка приложения:
make down
